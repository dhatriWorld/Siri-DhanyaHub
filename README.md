# 🌾 Siri-Dhanya Hub
A Millet Value Chain Android App for Karnataka

## Features
- 📈 Mandi Watch — Live millet prices with ▲▼ trends
- 🍲 Recipe Lab — Search + save millet recipes with step-by-step guide
- 💚 Health Benefits — Why millets are superfoods
- 🛒 Direct Buy — Connect with local FPOs

## Tech Stack
Android Studio | Java | Firebase | RecyclerView | Glide

## How to Run
1. Clone this repo
2. Add your own google-services.json to the app/ folder
3. Build and run in Android Studio
